# Beta Release Checklist

## 目的

- β公開前に、MVPが一般ユーザーに触ってもらえる最低条件を満たしているか確認する。
- 既知の制限を明文化し、MVP範囲外の期待値を管理する。
- 次フェーズの開発対象を整理する。
- 本ドキュメントは `docs/14_manual_qa_checklist.md` の実施後に確認する。

## 既知の制限

- ノックアウトステージは未投入。
- 全104試合はまだ揃っていない。
- 試合結果は手動更新前提。
- リアルタイム速報はない。
- LINE通知はない。
- ログイン機能はない。
- 課金機能はない。
- 自動X投稿はない。
- 外部API連携はない。
- スクレイピングはしない。
- 公式ロゴ・公式画像・公式風デザインは使わない。
- AIに順位計算や突破条件判定は任せない。
- 順位計算・3位通過ライン・重要度判定はTypeScriptロジックで行う。
- 解説文や共有文はMVPでは定型文中心。
- 詳細タイブレーク、直接対決、フェアプレーポイント、抽選条件は未反映。
- FIFA公式データの再確認は手動。
- FIFA公式以外の補助確認はReuters / AP / BBC / ESPN / 公式放送局 / 各国協会公式などに限定する。
- 一般ブログ、まとめサイト、転載系、SNS、Wikipediaは補助ソースに使わない。

## β公開時の表示・説明方針

- β版として扱う。
- 「W杯 観戦ナビ」は非公式の個人制作ツールである。
- ヘッダーで `2026 FIFA World Cup 対応` を表示する。
- 画面上でも `非公式の個人制作ツールです` を表示する。
- 日程ソースはFIFA公式を参照している。
- FIFAロゴや公式画像は使わない。
- 表示時刻は日本時間である。
- 日付と時刻は日本時間・曜日つきで表示する。
- 情報の最終確認日は画面下部に表示する。
- 共有文コピーは自動投稿ではなく、ユーザーが任意でXやLINEなどへ貼り付ける方式である。
- 結果・順位・突破条件はMVP版の簡易ロジックであり、公式順位表の代替ではない。
- 詳細条件は公式情報を確認する必要がある。
